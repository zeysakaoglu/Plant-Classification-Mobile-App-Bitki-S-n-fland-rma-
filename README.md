# Plant Classification Mobile Application

Mobile application that classifies plants as healthy or weed using a CNN-based deep learning model.

## Technologies
Flutter, Flask, TensorFlow, Firebase

## Features
- Capture or upload plant image
- AI-based classification
- Percentage-based result
- Mobile interface

## Project Type
Graduation Project
